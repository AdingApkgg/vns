е·ІзҹҘиҠіеҘ¶зў°дёҚеҲ°д»ҺйӣЁпјҢд»…жңүз”·дё»иғҪж‘ёеҲ°д»ҺйӣЁпјҢеҰӮжһңиҠід№ғе’Ңд»ҺйӣЁд»ҘзӣёеҗҢзҡ„е§ҝеҠҝйҮҚеҸ иө·жқҘпјҢз”·дё»жҳҜдёҚжҳҜиғҪеӨҹдёҖж¬Ўе•ӘдёӨдёӘпјҹ еҸҲеӣ дёәз”·дё»еҸӘжңүдёҖж №пјҢиҠід№ғе’Ңд»ҺйӣЁеҝ…йЎ»дҝқжҢҒй«ҳеәҰйҮҚеҸ зҡ„дҪҚзҪ®з”·дё»жүҚиғҪиҝӣе…ҘпјҢйҡҫд№ҲеңЁз”·дё»иҝӣе…ҘеҗҺпјҢиҠід№ғе’Ңд»ҺйӣЁзҡ„дҪҚзҪ®й”ҷејҖдјҡжҖҺд№Ҳж ·пјҹ

зӯ”пјҡ 1.еҪ“йёЎз«Ӣж–ӯпјҹ 2.е·ІзҹҘ жқЎд»¶пјҡиҠід№ғе’Ңз”·дё»жңүзў°ж’һиҜҶеҲ«

е№јеҲҖе’Ңз”·дё»жңүзў°ж’һиҜҶеҲ« е№јеҲҖе’ҢиҠід№ғж— зў°ж’һиҜҶеҲ«

зҺ°еңЁеӯҳеңЁзҡ„жғ…еҶөжҳҜпјҡ еҪ“иҠід№ғе’Ңе№јеҲҖзҡ„з©ҙеҸ еңЁдёҖиө·ж—¶ з”·дё»еҗҢж—¶иҝӣе…ҘдёӨдёӘз©ҙдёӯ

и§Јпјҡ жҲ‘д»¬д»Ҙз”·дё»дёҒдёҒдҪңдёәеҺҹзӮ№е»әз«Ӣз©әй—ҙеқҗж Үзі» жӯӨж—¶з”ЁеҸіжүӢе®ҡеҲҷ д»ҘдёҒдёҒзҡ„еүҚиҝӣж–№еҗ‘дёәXиҪҙ з”·дё»зҡ„и…°жӨҺе»¶з”іж–№еҗ‘дёәYиҪҙ з”·дё»и§Ҷи§’зҡ„еҸіжүӢжүҖеңЁжңқеҗ‘дёәZиҪҙжӯЈж–№еҗ‘

еҲҶи§ЈиҝҗеҠЁиҝҮзЁӢеҸҜзҹҘ еӯҳеңЁзҡ„иҝҗеҠЁж–№ејҸдё»иҰҒеҲҶдёӨз§Қ дёҖз§Қдёәе»¶XиҪҙдҪҚз§» еҸҰдёҖз§Қдёәз»•XиҪҙж—ӢиҪ¬ з”ұе·ІзҹҘеҫ—зӣёдә’дҪңз”Ёе…ізі» жҲ‘д»¬еҸҜд»Ҙе»әз«ӢдёҖдёӘжЁЎеһӢ иҠід№ғдёҺд»ҺйӣЁд№Ӣй—ҙдә’дёҚе№Іжү° еҸӘжңүйҒҮеҲ°з”·дё»жүҚдјҡеҸ‘з”ҹдҪңз”Ё иҝҷе°ұзӣёеҪ“дәҺиҠід№ғе’Ңе№јеҲҖеҲҶеҲ«дёәй»„иүІдёҺз»ҝиүІзҡ„жүӢз”өе…ү дёӨжқҹе…үжү“еңЁеўҷдёҠиў«жҲ‘д»¬зңӢи§ҒзӣёеҪ“дәҺдҪңз”ЁдәҺз”·дё»дёҒдёҒ иҝҷйҮҢеӯҳеңЁеўҷдёҠеӣӣз§Қжғ…еҶө

  1. д»…иў«й»„е…үз…§еҲ°
  2. д»…иў«з»ҝе…үз…§еҲ°
  3. еҗҢж—¶иў«й»„з»ҝе…үз…§еҲ°
  4. йғҪжІЎз…§еҲ°

иҝҷйҮҢеҜ№еә”иҝҗеҠЁдёӯзҡ„еӣӣз§ҚдҪңз”ЁеҢәеҲ«

  1. ж‘©ж“ҰиҠід№ғзҡ„з©ҙ
  2. ж‘©ж“Ұе№јеҲҖзҡ„з©ҙ
  3. еҗҢж—¶ж‘©ж“ҰдёӨдёӘз©ҙ
  4. йғҪжІЎеңЁж‘©ж“Ұ
